FAQ

01.
Is it a paid app?
No, Screenshoter+ is totally free. You don't have to pay to download, install or use the software. All features are available for free.
02.
What platforms can Zoom be used on?
Screenshoter+ is available for Windows only, but a macOS version will be released later.
03.
What's the difference between Screenshoter and other software?
The main advantage is that it's a 2-in-1 tool. You can take a screenshot or start screen recording with sound in one click. Just press both mouse buttons or a hotkey.
04.
Where are screenshots and recordings saved?
You can specify in the settings where to save screenshots and screen recordings. By default, files are saved in the local storage on your computer. You can also have the file sent to a server. As soon as you take a photo or record a video, you can share it with anyone via a link.
05.
Can I delete files from the server in a couple of clicks?
Yes, you can. You need to use the generated link, enter your User ID (you can copy it in the settings), and confirm on delete. It's convenient because you don't need to create an account for it.
